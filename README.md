# Personal Portfolio

A separate personal website project for Tongxue Xiao.

## Purpose

This project is intentionally separate from the `meow-cup-landing` product site.

It is meant to become a focused personal website for:

- professional positioning
- selected project presentation
- resume-friendly content
- contact and portfolio sharing

## Stack

- React
- Vite
- GSAP

## Run

```bash
npm install
npm run dev
npm run build
```
